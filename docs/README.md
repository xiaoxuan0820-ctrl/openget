# OpenGet - 提示词优化工具

## 📱 项目概述

**OpenGet** 是一款面向普通用户的提示词优化工具，旨在帮助用户将自然语言转换为更精准、专业的AI指令。通过智能拆解和术语转换，让每个人都能轻松获得高质量的AI交互体验。

## 🎯 核心价值

- **降低使用门槛**：用户无需学习复杂的提示词技巧
- **提升输出质量**：结构化的提示词带来更准确的AI回复
- **简化创作流程**：一键将模糊想法转化为专业指令

## 🔧 核心功能

### 1. 结构化拆解
将用户的模糊想法自动拆解为以下清晰模块：
- **目标**：明确用户的核心意图
- **背景**：补充必要的上下文信息
- **要求**：具体的约束条件和标准
- **输出格式**：期望的结果形式

### 2. 专业术语转换
- 将口语化表达转换为专业、规范的AI可理解语言
- 自动补充缺失的必要信息
- 优化表达结构，提升指令清晰度

## 📋 API配置

- **Endpoint**: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
- **Model**: `glm-4-flash`
- **Key**: 配置于 `config/api_config.dart`

## 🔄 提示词优化流程

```
用户输入 → 结构化分析 → GLM处理 → 格式化输出
   ↓           ↓           ↓          ↓
模糊想法 → 拆解模块 → 智能优化 → 完整提示词
```

### System Prompt 设计

核心指令让GLM输出包含：
1. 【目标】- 明确意图
2. 【背景】- 上下文信息
3. 【要求】- 约束条件
4. 【输出格式】- 期望结果
5. 【优化后的完整提示词】- 最终可用指令

## 📄 功能流程

### 主界面交互
1. 用户在输入框输入自然语言需求
2. 点击「优化」按钮触发处理
3. 显示加载状态
4. 展示结构化分析结果
5. 支持一键复制优化后的提示词

### 状态处理
- **空状态**：引导用户输入需求
- **加载状态**：显示加载动画
- **成功状态**：展示优化结果
- **错误状态**：显示错误信息，提供重试选项

## 🔒 安全与隐私

- API密钥存储于本地配置文件
- 不上传用户输入到第三方服务
- 优化过程完全在本地+API端完成
